# Flink Job执行流程

flink run -> CliFrontend.run -> 解析命令行参数，构造PackagedProgram和环境配置 -> 执行main方法


Job 构建
1、 构建ExecutionEnvironment
2、 添加算子（source，sink，operater等）
3、 执行execute

execute yarn流程
1、 构建Pipeline
2、 根据配置构建Executor(Yarn、Local、K8S、Remote等)
3、 执行Pipeline 
  3.1、 转换 Pipeline -> JobGraph
  3.2、 部署到Yarn集群
    3.2.1 检查Yarn队列资源
